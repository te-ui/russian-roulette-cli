
# Russian Roulette CLI Game

A simple 2-player text-based Russian Roulette game in Python.  
Safe, fun, and turn-based. First player to get "Bullet hit" loses.

## Features

- 2-player turn-based game
- Randomized “Bullet hit” vs “No bullet”
- Quit anytime by typing 'q'
- Console-based (no graphics)
- Fully safe — just text
